# Zayed Dev Snippets

A powerful snippet collection for modern React development.

## Prefixes

| Prefix | Technology |
|--------|-----------|
| `ts.` | TypeScript |
| `tw.` | Tailwind CSS |
| `rr.` | React Router |
| `fm.` | Formik |
| `yp.` | Yup |
| `rsb.` | Storybook |

## Installation

Install from VSIX or copy to `.vscode/extensions/`
